# Example-InputTest

Ein Testprogramm für die Terminal-Eingabe-Funktionalität des xAkiitoh Program Launchers.

## Was testet dieses Programm?

- **Benutzername-Eingabe**: Fragt nach dem Namen des Benutzers
- **Mikrofon-Auswahl**: Simuliert eine typische Mikrofon-Auswahl (1-4)
- **Streaming-Einstellungen**: Ja/Nein-Abfrage mit Auflösungs-Wahl
- **Interaktive Befehle**: Während der Laufzeit können `stop` oder `status` eingegeben werden

## Verwendung

1. Starte das Programm über den Program Launcher
2. Beantworte die Fragen im Terminal-Eingabefeld
3. Nutze die Eingabe-Funktionalität für verschiedene Szenarien

## Beispiel-Eingaben

1. **Name**: `xAkiitoh`
2. **Mikrofon**: `1` (für Hauptmikrofon)
3. **Streaming**: `ja`
4. **Auflösung**: `2` (für 1080p)
5. **Interaktive Befehle**: `status` oder `stop`

Dieses Programm demonstriert alle wichtigen Eingabe-Szenarien, die in echten Streaming-Programmen auftreten können.
